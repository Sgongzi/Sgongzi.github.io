拟合优度检验

检验拟合优度通常是考察一个来自某个未知分布的随机样本，检验其未知分布函数是否符合零假设为某个已知而具体的分布。

Kolmogorov-Smirnov检验(⭐)

使用一组数据的经验分布函数s(x)作为分布函数F(x)的估计是一种直观的做法。我们把S（x）与F（x）的最大垂直距离函数的统计量称为Kolmogorov型统计量，把两个经验分布函数最大垂直距离函数的统计量称为Smirnov型统计量。

np.random.seed(0)
sample_data = np.random.normal(loc=0, scale=1, size=100)

ks_statistic, p_value = stats.kstest(sample_data, 'norm')

print(f"KS Statistic: {ks_statistic}")
print(f"P-value: {p_value}")

Shapiro-Wilk检验

这是一种用于检测样本数据是否来自正态分布的统计检验方法。适用于小样本数据，是一种常用的正态性检验方法。Shapiro-Wilk检验的基本思想是通过将样本数据的顺序统计量与正态分布的期望值进行比较，从而判断数据的正态性。

import numpy as np
from scipy.stats import shapiro

data = [4.4, 5.5, 6.3, 7.1, 7.6]
stat, p_value = shapiro(data)
print('Shapiro-Wilk 检验统计量: ', stat)
print('p值: ', p_value)

符号检验(⭐⭐)

符号检验将数据分为大于检验量的部分和小于检验值的部分，得到正符符号之和为检验统计量，这个统计量显然是一个二项分布，然后采用二项分布的检验即可。

步骤

  1. 样本从小到大排序；
  2. 计算大于检验值的部分的数量Y=∑inI(Xi>x0)Y = \sum_i^nI(X_i>x_0)Y=∑in​I(Xi​>x0​)作为检验统计量；
  3. 检验统计量服从二项分布Y b(n,1−p)Y ~ b(n,1-p)Y b(n,1−p)；
  4. 得到拒绝域，得到检验结果；
import numpy as np
from scipy.stats import binom_test
data = np.array([4, 5, 6, 7, 5, 4, 8, 6, 3, 5])
m0 = 5
signs = np.sign(data - m0)

n_pos = np.sum(signs > 0)
n_neg = np.sum(signs < 0)

p_value = binom_test([n_pos, n_neg], alternative='two-sided')

print('正号数量: ', n_pos)
print('负号数量: ', n_neg)
print('p值: ', p_value)

Wilcoxon符号秩模型(⭐⭐)

符号检验利用观察值和零假设的中心位置之差的符号来进行检验，但是它并没有利用这些差的大小（体现于差的绝对值的大小）所包含的信息。不同的符号代表了在中心位置的哪一边，而差的绝对值的秩的大小代表了距离中心的远近。

  1. 对每个样本量，计算其与检验值的距离；
  2. 对得到的距离从小到大排序，得到秩；
  3. 赋予每个秩正负号（比检验值大还是比检验值小）；
  4. 得到正的符号秩的和作为检验统计量；
  5. 得到拒绝域；

from scipy.stats import wilcoxon
data = [6, 7, 5, 6, 7, 8, 5, 6, 4, 5]
m0 = 5
differences = [x - m0 for x in data]


stat, p_value = wilcoxon(differences)

print('Wilcoxon 符号秩检验统计量: ', stat)
print('p值: ', p_value)

游程检验(⭐⭐)

游程检验模型 假定我们掷一个硬币，1/2得到正面（记为1），1/2到反面（记为0），作为一个Bernoulli试验。如果这个试验是随机的，则不大可能出现许多1或许多0连在一起，那么如何检验一个数据是否是随机的呢？可以使用游程检验。

一个游程指的是一组数据连续相同元素的最大区间。

游程检验的步骤

  1. 将样本分为0，1两组（与中位数比大小等方法）；
  2. 计算游程数；例如 1011001011 的游程数为7；游程数为检验统计量
  3. 得到拒绝域，求得结果；
import numpy as np
from scipy.stats import norm

data = [1, 0, 1, 1, 0, 0, 1, 0, 1, 1]

runs = 1
for i in range(1, len(data)):
    if data[i] != data[i-1]:
        runs += 1

n1 = sum(data)
n2 = len(data) - n1
n = len(data)

# 期望游程数和标准差
E_R = (2 * n1 * n2) / n + 1
sigma_R = np.sqrt((2 * n1 * n2 * (2 * n1 * n2 - n)) / (n**2 * (n - 1)))

Z = (runs - E_R) / sigma_R
p_value = 2 * (1 - norm.cdf(abs(Z)))

游程检验判断两总体是否同分布(⭐)

思想：将两个样本X和Y从小到大混合排列，然后得到混合样本W，混合样本中属于X的记为1，属于Y的记为0，然后计算游程数，进行检验。

两样本位置参数检验

Wilcoxon秩和检验(⭐⭐)

思想：Wilcoxon秩和检验通过比较两个样本的秩来确定它们是否来自同一分布。具体来说，它比较两个样本的秩和，如果两样本同均值，那么理论上两样本的混合样本的秩和数据应该相近。

计算过程

  1. 混合样本X和样本Y得到混合样本；
  2. 对混合样本从小到大排序，得到秩；
  3. 求两样本的秩和，以样本量小的一方的秩和为检验统计量；
  4. 得到拒绝域，得出结果；
import numpy as np
from scipy.stats import mannwhitneyu

sample1 = [12, 15, 14, 10, 13, 15, 14, 16, 18]
sample2 = [20, 22, 25, 27, 24, 23, 19, 21, 26]
stat, p_value = mannwhitneyu(sample1, sample2)

print(f"U Statistic: {stat}")
print(f"P-value: {p_value}")

Siegel-Tukey检验 (⭐⭐)

如果一个总体方差较大，其样本点一定散布得较远。因此，这里的秩不是按大小来定，而是按散布远近而定。先把两个样本混合并按升幂排序，然后定最小的一个秩为1，然后把最大和次大的两个数的秩定为2和3，再回到小端定第二、第三小的秩为4，5，…，如此从一端跳到另一端，每端按从外到内的顺序取两个秩，直到所有的点都分配了秩为止。

步骤

  1. 混合样本，由小到大排序；
  2. 最小的数的秩为1，最大的数的秩为2，次大的数的秩为3，第二小的数秩为4….
  3. 求两样本的秩和；
  4. 得到拒绝域，求结果；

Ansari-Bradley检验(⭐)

混合样本后，将最大最小的秩都取1，其次都取2，以此类推。求出两样本分别的秩和。注意这个统计检验量在大样本情况下，也渐进服从正态分布。

Fligner-Killeen检验(⭐⭐)

（需要求混合样本中位数，需要得到到中位数的距离；）

思想：按照样本到中位数的大小，定义秩。

  1. 计算混合样本的中位数；
  2. 计算每个数到中位数的距离；
  3. 得到按照距离大小排序的秩；
  4. 求秩和，统计检验量即为样本X或者样本Y在距离样本中的秩和；

秩平方和检验(⭐)

前提：不要求位置参数相等

  1. 计算两样本到各自均值的距离，得到dx,dyd_x,d_ydx​,dy​
  2. 混合两样本的距离，使用距离定义秩
  3. 求两样本的秩的平方和，即为检验统计量；

随机设计数据:Jonckheere-Terpstra(⭐⭐)

JT检验是一种非参数检验，用于检验多个组的中位数是否存在单调趋势。其主要用于分析变量之间是否存在某种方向性的排列趋势。

检验统计量为：第i组样本中的观测值小于第k组样本中的观测值的个数；记为

J=∑1<=i<k<=rUikJ = \sum_{1<=i<k<=r}U_{ik}J=1<=i<k<=r∑​Uik​

例题：

import numpy as np
from scipy.stats import rankdata, spearmanr

def jonckheere_terpstra(groups):
    k = len(groups)
    n = sum(len(group) for group in groups)
    r = rankdata(np.concatenate(groups))
    jt_stat = 0
    
    for i in range(k - 1):
        for j in range(i + 1, k):
            group_i = r[np.sum([len(groups[m]) for m in range(i)]): np.sum([len(groups[m]) for m in range(i + 1)])]
            group_j = r[np.sum([len(groups[m]) for m in range(j)]): np.sum([len(groups[m]) for m in range(j + 1)])]
            for gi in group_i:
                for gj in group_j:
                    if gi < gj:
                        jt_stat += 1

    return jt_stat



group1 = [2, 3, 5]
group2 = [4, 6, 7]
group3 = [8, 9, 10]

jt_stat = jonckheere_terpstra([group1, group2, group3])
print(f"Jonckheere-Terpstra statistic: {jt_stat}")
