游程

maths notes

游程检验（Runs Test）是一种非参数统计检验方法，主要用于检验一组二元数据（如正负、男女、成功失败等）是否随机排列。游程检验的关键概念是“游程”（run），即一连串相同类型的元素。总游程数指的是数据序列中连续相同元素的区段数。

游程（Run）的定义

一个游程（run）是指数据序列中连续相同元素的最大区段。例如，对于序列 \([+, +, -, -, +, -, +, +]\)，游程包括：

  1. \([+, +]\) （第一个正号游程）
  2. \([-]\) （第一个负号游程）
  3. \([-]\) （第二个负号游程）
  4. \([+]\) （第二个正号游程）
  5. \([-]\) （第三个负号游程）
  6. \([+, +]\) （第三个正号游程）

总游程数就是上述游程的个数，即 6。

def count_runs(sequence):
    if len(sequence) == 0:
        return 0

    runs_count = 1

    for i in range(1, len(sequence)):
        if sequence[i] != sequence[i - 1]:
            runs_count += 1

    return runs_count

# 示例序列
sequence = ['+', '+', '-', '-', '+', '-', '+', '+']
print(f"总游程数: {count_runs(sequence)}")
