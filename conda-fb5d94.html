conda基础操作

CS

1.查看所有环境

conda env list

conda info -e

2.创建新的环境

conda create -n env_name python=3.8

3.激活环境

conda activate env_name

4.退出环境

conda deactivate

5.删除环境

conda remove -n env_name -all

6.安装包

conda install pkg_name -n env_name
conda uninstall pkg_name -n env_name

7.升级包

conda update pkg_name

8.搜索包

conda search pkg_name

9.指定镜像源

pip install -c conda-forge pkg_name

10.更改镜像源（清华）

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/cloud

11.列出安装的包

conda list

12.导出环境到yml文件

conda env export > environment.yml

13.利用yml文件配置环境

conda env create -f environment.yml
